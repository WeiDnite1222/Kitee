# Kitee
A custom Minecraft launcher prototype (Forked from BakeLauncher)

# How to use

Make sure you have installed `uv` before you do the process below.

##### Sync project with command: 

`uv sync`

##### Run launcher:

`uv run -m kitee_launcher.main`
